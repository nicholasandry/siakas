## Ringkasan
SIAKAS (Sistem Informasi Aset Keuskupan Surabaya) adalah sistem pencatatan aset, unit, badan hukum, dan audit trail untuk Keuskupan Surabaya.

## Teknologi
- Frontend / fullstack: Next.js
- ORM: Drizzle
- Database: PostgreSQL
- UI components: shadcn/ui
- Icons: lucide-react
